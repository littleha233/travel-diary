# TravelAround

TravelAround v0.1 is an Expo + React Native + TypeScript static frontend prototype for a travel check-in and travel map app.

## Tech Stack

- Expo
- React Native
- TypeScript
- Expo Router
- StyleSheet
- Local mock data

## Getting Started

Install dependencies:

```bash
npm install
```

Start the Expo dev server:

```bash
npm start
```

Run on iOS, Android, or Web:

```bash
npm run ios
npm run android
npm run web
```

Run TypeScript checks:

```bash
npm run typecheck
```

## Prototype Scope

This phase uses mock data only. It does not connect to a real backend, real map SDK, real GPS location, real login, real image upload, or real AI generation.

Implemented screens include:

- Login / guest entry
- 5 bottom tabs: Home, Plan, CheckIn, Community, Profile
- City detail
- Spot detail
- Plan detail
- Theme quest detail
- Trip detail
- AI memory generation/detail
- Achievements
- Settings
